
Steps on how to build/compile project:

1. Clone ToyRobot Repository - https://github.com/Jonjon1028/ToyRobot.git

2. Open the ToyRobot.sln in Visual Studio.

3. Go to ToyRobot project and select Program.cs and press F5 to run project.

4. You can edit the position inside the Program/Main method for robot's placement and then Press F5 to run project.
